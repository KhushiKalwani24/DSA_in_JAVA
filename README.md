## 📖 About

This repository is my personal **DSA learning journal** — a growing collection of Data Structures and Algorithms implemented from scratch in Java. Each file represents a concept studied, a problem solved, and a step forward in thinking like a better programmer.

> 🔄 **This repo is actively updated** as I progress through DSA topics. Star it to follow along!

## 🚀 How to Run

### Prerequisites
- Java JDK 8 or higher installed
- Any IDE (IntelliJ IDEA / VS Code / Eclipse) or terminal

### Run any file

```bash
# Clone the repo
git clone https://github.com/KhushiKalwani24/DSA_in_JAVA.git
cd DSA_in_JAVA

# Compile
javac BinarySearch.java

# Run
java BinarySearch
```
